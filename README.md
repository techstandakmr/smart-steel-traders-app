# Smart Steel Traders - Shop Management Mobile App

A complete **Shop Management System** built for **Smart Steel Traders** to manage inventory, sales, purchases, transactions, customers, suppliers, and reporting efficiently.

This application is designed for shop owners and sales staff to streamline daily business operations through a mobile-first solution.

---

## Tech Stack

### Mobile Application
- React Native

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## Project Overview

This app helps manage:

- User authentication & role-based access
- Product inventory
- Categories
- Purchase management
- Sales management
- Suppliers & customers
- Payment transactions
- Reports & profit analytics

---

## User Roles

### 1. Admin
Admin has full access to all modules.

Capabilities:

- Manage all users
- Manage products
- Manage categories
- Manage suppliers
- Manage customers
- View reports
- Manage transactions
- Full system control

---

### 2. Salesman
Salesman has limited operational access.

Capabilities:

- Create sales
- Manage orders
- View assigned data
- Generate reports
